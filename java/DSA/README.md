# Data Structures and Algorithms (DSA) Repository

This repository contains Java implementations of various data structures and algorithms, organized into categories for easy navigation and understanding.

## 📁 Folder Structure

### LinkedList/
- **DLL.java**: Doubly Linked List implementation with insertion and deletion operations
- **SinglyLinkedList.java**: Singly Linked List with insert, delete, and display operations
- **LinkedListUtility.java**: Utility functions for linked lists including frequency counting and duplicate removal
- **LinkedListStringOps.java**: String operations on linked lists (reverse and sort)

### Stack/
- **ArrayStack.java**: Array-based stack implementation with menu-driven interface
- **LinkedListStack.java**: Stack implementation using linked list
- **InfixPostfixConversion.java**: Conversion of infix expressions to postfix using stack

### Queue/
- **CircularQueue.java**: Circular queue implementation using arrays
- **Dequeue.java**: Double-ended queue (deque) implementation
- **PriorityQueue.java**: Priority queue with proper ordering

### Tree/
- **BinarySearchTree.java**: Complete binary search tree with insert, search, and traversals
- **BSTMenuDriven.java**: Menu-driven binary search tree operations
- **TreeTraversal.java**: Tree traversal algorithms (inorder, preorder, postorder) with level-order insertion
- **ThreadedBinaryTree.java**: Threaded binary tree implementation for efficient traversals

### Graph/
- **GraphAdjacencyList.java**: Graph implementation using adjacency list with DFS and BFS traversals

### Applications/
- **Polynomial.java**: Polynomial addition and operations using linked lists
- **PostfixEvaluation.java**: Evaluation of postfix expressions
- **Calculator.java**: Basic calculator implementation
- **MatrixMultiplication.java**: Matrix multiplication operations

## 🚀 Features

- **Comprehensive Coverage**: Implements fundamental data structures and algorithms
- **Menu-Driven Interfaces**: Many implementations include interactive menus for testing
- **Clean Code**: Well-structured, readable Java code with proper naming conventions
- **Educational Purpose**: Suitable for learning and understanding DSA concepts

## 🛠️ Usage

Each Java file contains a `main` method for testing the implementation. Compile and run individual files:

```bash
javac LinkedList/DLL.java
java LinkedList.DLL
```

## 📚 Concepts Covered

- **Non-Linear Data Structures**: Trees, Graphs
- **Algorithms**: Searching, Sorting, Traversals, Expression Evaluation
- **Advanced Topics**: Threaded Trees, Priority Queues, Graph Traversals
